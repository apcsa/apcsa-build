Programming objects often represent, or model, real-world objects.

This course is about a style of programming called object-oriented programming. This term implies a focus on designing programming objects. The most obvious kind of object that a programmer might write is one that models something in the real world: that is, one that behaves like the thing in the real world. By working with the programming object, we may find out information about its real-world counterpart.

An alarm clock is an example of a thing one can represent, or model, as a programming object. To make a program that instructs the computer to act like an alarm clock, one would make a programming object that acts like an alarm clock. All programming objects have names. A good name for this object would be alarm clock.
